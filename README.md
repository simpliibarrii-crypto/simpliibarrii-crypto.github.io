# Barry Clerjuste · AI Engineering Portfolio

A dependency-light GitHub Pages portfolio for showcasing local-first AI systems, applied research, and interactive engineering demonstrations.

## Public surfaces

- `index.html` — professional portfolio, project filters, engineering principles, and contact surface
- `research.html` — working papers, engineering reports, implementation studies, and research method
- `project.html?project=<id>` — reusable case-study page with deterministic browser demos
- `app.js` — project registry, navigation, filters, and seven interactive demos
- `styles.css` — responsive obsidian, crimson, and champagne design system
- `404.html` — graceful branded error state

## Demonstrated systems

1. Home for AI — agent orchestration trace
2. Raven AI — evidence graph builder
3. OpenClinical AI — consent-gated shift handoff
4. Hermes Edge — device-aware routing policy
5. Raven BioComputer — deterministic DNA sequence tools
6. JSpace Chain — observable workspace and risk decision
7. LangChain NVIDIA study — agentic RAG route planner

The demos execute locally in the browser and are deliberately scoped. They do not require API keys, patient data, model downloads, or a running production backend.

python3 -m http.server 8000
